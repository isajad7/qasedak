# Productization P6 - License Decision

No `LICENSE` file is present yet, so no license has been chosen in this phase.

Choose one before any public GitHub release:

| Option | Practical effect |
| --- | --- |
| MIT | Short permissive license; allows commercial/private reuse with attribution. |
| Apache-2.0 | Permissive license with explicit patent terms. |
| AGPL-3.0 | Strong copyleft for network services; public service modifications generally need source availability. |
| Proprietary/private | No public open-source grant; suitable for private GitHub or commercial-only distribution. |

```text

```
